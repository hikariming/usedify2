欢迎来到 **从零开始的 Dify 智能体教程**！本路线图帮助你了解各个模块之间的衔接关系，方便规划学习节奏。

## 你将构建什么

完成全部课程后，你将搭建一个可上线的智能体工作区：

1. 集成 Dify 工作流，支持自动工具调用的智能体。
2. 构建包含数据集、反馈信号与安全护栏的评估闭环。
3. 通过 API 网关上线，并接入使用统计与成本控制。

## 模块概览

| 模块 | 主题 | 收获 |
| ---- | ---- | ---- |
| 环境搭建 | 工具链、代码仓库、质量保障 | 可复现的本地 + 云端开发环境 |
| 生命周期 | 项目脚手架与领域建模 | 适合团队协作的架构实践 |
| 上线部署 | 可观测性、扩展性、合规 | 自信地对外发布智能体 |

展开左侧目录即可进入下一节，或继续阅读先决条件。

## 前置要求

- 具备 TypeScript / Python 基础。
- 拥有一套 Dify 服务（云端或自建）及 API 凭证。
- 熟悉 Git 工作流。
