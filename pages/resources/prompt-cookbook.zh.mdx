---

以下范式可作为起点，根据领域知识进一步调优。

## 安全感知型调研智能体

```yaml
角色: research-analyst
风格: 分析型，附带引用
流程:
  - 通过 web-search 收集上下文
  - 将结论整理为要点列表
  - 以 Markdown 表格列出引用来源
约束:
  - 无来源时拒绝回答
  - 置信度 < 0.6 时主动追问补充信息
```

## 客服分诊助手

```yaml
意图识别: classification(prompt="工单意图", labels=["billing", "technical", "other"])
回复模板: |
  您好 {{name}}，
  感谢反馈关于 {{intent}} 的问题。
  {{solution}}
  若仍未解决，请回复 “Handoff” 以转人工客服。
```

建议将提示词纳入版本控制，并在发布时打上标签，确保团队回溯历史。
